## 风险提示

| 风险 | 建议 |
|------|------|
| 前后端各 1 人，任务阻塞无缓冲 | 优先 P0 链路，P2 可砍 |
| MinIO 集成可能踩坑 | Sprint 3 提前验证 MinIO Docker 部署 |
| 响应式适配工作量大 | 移动端优先，桌面端渐进增强 |
| 敏感词库质量影响合规 | 导入 GitHub 开源词库做基础，人工补充 |
